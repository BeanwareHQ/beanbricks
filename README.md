## brickout.c

A single-file breakout clone in C and Raylib.

## Dependencies

 * Raylib
 * Raygui (bundled)

## Building

 * `make`

## Running

 * `./brickout`

## Making a tarball

 * `make tarball`

## Configuring

Go through the options and comments found in `settings.h`. Change the `#define`s as you wish, and recompile by running `make`.
